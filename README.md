# 60 Days of Python 🐍

A personal challenge to write Python every single day for 60 days straight — starting from the absolute basics and working up through data structures, functions, and real-world mini-projects.

This repo is my public learning log. Each file maps to one day, one topic, one step forward.

---

## Why I did this

Learning something properly takes repetition. I wanted a structure that forced me to show up daily, write actual code, and build on what I learned the day before — not just watch tutorials and forget everything a week later.

---

## What's covered

The challenge moves through Python in a natural progression:

**Foundations (Days 1–8)**
Variables, strings, data types, operators, and conditionals — the stuff everything else is built on.

**Loops & Control Flow (Days 9–11)**
`for` loops, `while` loops, nested loops, `break`, and `continue`.

**Data Structures (Days 12–21)**
Lists, list methods, list comprehension, tuples, sets, frozen sets, arrays, and dictionaries — each with their own dedicated day.

**Built-in Tools (Days 22–28)**
`zip`, `enumerate`, search and replace, and a tour through Python's built-in functions.

**Functions (Days 29–38)**
Defining functions, return types, parameters vs arguments, `*args` and `**kwargs`, recursion, lambda expressions, and `map`/`filter`/`reduce`.

**Mini Projects (Day 40+)**
Putting it all together — including a YouTube downloader built from scratch.

---

## File structure

Each file follows the naming pattern `day_N(topic).py`, so it's easy to jump to whatever you're looking for.

```
day_1(basic).py
day_12(lists).py
day_36(recursion).py
day_40(yt_downloader).py
...
```

---

## How to run any file

You just need Python 3 installed. Then:

```bash
git clone https://github.com/root-rashed/60-days-of-python.git
cd 60-days-of-python
python day_1\(basic\).py
```

For the YouTube downloader (Day 40), install the dependency first:

```bash
pip install pytube
python day_40\(yt_downloader\).py
```
